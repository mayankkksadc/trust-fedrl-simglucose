# Trust-FedRL over Dynamic Trust Networks (SimGlucose Case Study)

This repository contains experiments for:
- Local RL baseline (PPO)
- FedAvg-based federated RL baseline
- Trust-weighted federated RL (Trust-FedRL)
using SimGlucose as the glycemic control case study.

## Structure
- `notebooks/`: Colab notebooks for running experiments
- `src/`: reusable environment + FL/RL code
- `paper/`: LaTeX draft
